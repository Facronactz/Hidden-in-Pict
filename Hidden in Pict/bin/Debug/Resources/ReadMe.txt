            |  # About:                                                                                            |
            |  -------------------------------------------------------------------------------------------------   |
            |                                                                                                      |
            |  - Hidden in Pict.bat                                                                                |
            |    Hide archive file inside the picture file                                                         |
            |                                                                                                      |
            |  - This program contains following features:                                                         |
            |    > Hide archive inside picture                                                                     |
            |    > Detect hiden archive inside picture                                                             |
            |                                                                                                      |
            |                     ## You can add password in rar file manually with WinRAR ##                      |
            |                                                                                                      |
            |         *** TO VIEW HIDDEN ARCHIVE INSIDE THE PICTURE JUST OPEN THE PICTURE WITH WinRAR ***          |
            |                                                                                                      |
            |  -------------------------------------------------------------------------------------------------   |
            |  # How to Input File:                                                                                |
            |  -------------------------------------------------------------------------------------------------   |
            |                                                                                                      |
            |  - Not all format supported                                                                          |
            |    > Archive: Archive just support only WinRAR archive (.rar)                                        |
            |    > Picture: Picture support PNG and JPG (*other format is still on work)                           |
            |  - Multiple file is not supported                                                                    |
            |  - You can drag and drop any file (Recommended)                                                      |
            |  - The archive file must be created before                                                           |
            |  - This program will detect if there any archive inside picture file                                 |
            |  - The picture file will work completely same as before (*you can open or even edit it)              |
            |  - If you want to mannualy write the file please include the complete location of the file           |
            |    > ex: "C:\Users\USER\Downloads\file.txt"                                                          |
            |  - Please dont add quote, it will make the program crash                                             |
            |                                                                                                      |
            |  -------------------------------------------------------------------------------------------------   |
            |  # Add content to archive inside picture file                                                        |
            |  -------------------------------------------------------------------------------------------------   |
            |  - Open the picture with WinRAR                                                                      |
            |  - Locate the file which will be added                                                               |
            |  - Drag the file and drop it in WinRAR                                                               |
            |  - It will automatically added the file and update the archive                                       |
